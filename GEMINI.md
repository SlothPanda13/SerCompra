# Gemini CLI en SerCompra

Este archivo sirve como punto de referencia para la interacción con Gemini CLI en el proyecto **SerCompra**.

## Descripción del Proyecto
**SerCompra** es una aplicación web desarrollada con ASP.NET Core MVC. El proyecto incluye:
- **Modelos:** Definiciones de base de datos (Entity Framework) en `SerCompra/Models/DataBase/`.
- **Controladores:** Lógica de negocio en `SerCompra/Controllers/`.
- **Vistas:** Plantillas Razor en `SerCompra/Views/`.
- **Base de Datos:** Scripts SQL en `SerCompra/MySqlDB/`.
- **Estilos y Scripts:** Recursos estáticos en `SerCompra/wwwroot/`.

## Cómo puedo ayudar
Puedo asistirte en diversas tareas, tales como:
- **Desarrollo de Funcionalidades:** Implementar nuevos controladores, modelos o vistas.
- **Depuración:** Analizar errores y proponer correcciones.
- **Refactorización:** Mejorar la estructura y calidad del código existente.
- **Documentación:** Generar explicaciones o comentarios para el código.
- **Consultas SQL:** Ayudar con la estructura de la base de datos en MySQL.

## Comandos Útiles
Para interactuar conmigo, puedes usar comandos naturales. Por ejemplo:
- "Analiza el controlador AccessController y dime si es seguro."
- "Crea una nueva vista para el perfil del Proveedor."
- "Explica cómo funciona la relación entre `Compra` y `BienservicoCompra`."
